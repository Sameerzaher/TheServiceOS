## מה נבנה?

הפורטל ללקוחות של ServiceOS הושלם במלואו! זהו מערך מלא של כלים המאפשר ללקוחות לנהל את התורים והתשלומים שלהם באופן עצמאי.

---

## ✨ 7 תכונות מרכזיות

### 1️⃣ הרשמה ואימות אימייל
- הרשמה עצמאית עם אימות חוזק סיסמה
- שליחת אימייל אימות אוטומטי
- דף אימות עם אפשרות לשליחה חוזרת

### 2️⃣ מערכת הזמנות
- מורים יכולים לשלוח הזמנות ללקוחות
- לקוחות מקבלים אימייל ובוחרים סיסמה
- החשבון מאומת אוטומטית

### 3️⃣ שכחת סיסמה
- בקשת איפוס סיסמה דרך אימייל
- טוקן חד-פעמי בתוקף שעה
- איפוס מאובטח עם אימות חוזק סיסמה

### 4️⃣ עריכת פרופיל
- עדכון שם, אימייל, טלפון
- שינוי סיסמה (עם אימות סיסמה נוכחית)
- עדכון אימייל דורש אימות מחדש

### 5️⃣ קביעת תור חדש
- בחירת מורה, תאריך, שעה ומשך
- בדיקת זמינות אוטומטית
- הוספת הערות אישיות

### 6️⃣ שינוי וביטול תורים
- שינוי מועד תור עתידי
- ביטול תורים בקלות
- צפייה בתורים עתידיים ועבר

### 7️⃣ היסטוריית תשלומים וחשבוניות
- סטטיסטיקות: שולם, ממתין, סה"כ
- הורדת חשבוניות HTML/PDF
- חשבוניות מקצועיות מעוצבות

## 🚀 איך להתחיל?

```sql

-- Run migrations 013 and 015
```

### 2. הגדר משתני סביבה
```env
# .env
NEXT_PUBLIC_APP_URL=http://localhost:3000
EMAIL_PROVIDER=console
RESEND_API_KEY=your_key  # אופציונלי
EMAIL_FROM=noreply@serviceos.com
```

### 3. הרץ את האפליקציה
```bash
npm run dev
```

### 4. פתח דפדפן
```
http://localhost:3000/client-signup
```
