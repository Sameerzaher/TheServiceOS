## 🎯 מה כלול?

### 1. ✅ Database Schema
- שדות authentication בטבלת `clients`
- טבלת `client_sessions` לסשנים
- טבלת `client_invitations` להזמנות
- Indexes + RLS policies

### 2. ✅ Authentication System
- שירות `clientAuth.ts`
- hash/verify סיסמאות
- יצירת טוקנים מאובטחים
- ניהול סשנים

### 3. ✅ API Endpoints
- `/api/client-portal/auth/login` - התחברות
- `/api/client-portal/me` - פרטי לקוח
- `/api/client-portal/appointments` - תורים
  - GET - שליפת תורים
  - DELETE - ביטול תור

### 4. ✅ UI Pages
- `/client-login` - דף התחברות
- `/client-dashboard` - דשבורד אישי
- סטטיסטיקות
- רשימת תורים עתידיים
- היסטוריית תורים

### 5. ✅ Features
- התחברות מאובטחת
- צפייה בתורים
- ביטול תור
- סטטיסטיקות
- Logout

## 🚀 התקנה

### 1️⃣ הרץ Migration

```bash
# ב-Supabase Dashboard:
# SQL Editor → העתק מ: 015_client_portal.sql → Run
```

### 2️⃣ Restart השרת

```bash
npm run dev
```

### 3️⃣ נסה!

**צור לקוח עם אימייל וסיסמה:**
```sql
-- ב-Supabase SQL Editor:
UPDATE clients 
SET 
  email = 'test@example.com',
  password_hash = '$pbkdf2-sha512$i=100000$...' -- hash של "Test123"
WHERE phone = '+972501234567';
```

**או השתמש בסקריפט:**
```bash
node scripts/create-client-account.js
```

## 🔧 מה חסר? (20% נותרים)

### תכונות שלא הספקנו:
1. ❌ **הרשמה (Signup)**
   - דף הרשמה
   - API endpoint
   - אימות אימייל

2. ❌ **הזמנת תור חדש**
   - בחירת תאריך ושעה
   - בחירת סוג שירות
   - תשלום

3. ❌ **שינוי תור (Reschedule)**
   - בחירת תאריך חדש
   - אישור מהמורה

4. ❌ **היסטוריית תשלומים**
   - כל התשלומים
   - הורדת חשבוניות PDF

5. ❌ **עדכון פרטים**
   - שינוי שם/טלפון
   - שינוי סיסמה
   - העדפות התראות

6. ❌ **הזמנות (Invitations)**
   - מורה שולח הזמנה ללקוח
   - לקוח לוחץ על קישור
   - מגדיר סיסמה

7. ❌ **Forgot Password**
   - איפוס סיסמה ללקוחות
