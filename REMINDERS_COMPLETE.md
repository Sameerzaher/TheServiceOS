## 💰 לשליחה אמיתית - Twilio

### למה Twilio?
- ✅ $15 קרדיט חינם בהרשמה
- ✅ תומך ב-SMS וב-WhatsApp
- ✅ 1,000 הודעות WhatsApp חינם לחודש!
- ✅ אמין ומקצועי

### מחירים
- **WhatsApp**: 1,000 חינם, אחר כך 2 אגורות להודעה
- **SMS**: ~30 אגורות להודעה

### התקנה מהירה

1. **הירשם**: https://www.twilio.com/try-twilio
2. **קבל פרטים**: Account SID + Auth Token
3. **עדכן `.env`**:
   ```bash
   MESSAGING_PROVIDER=twilio
   TWILIO_ACCOUNT_SID=your_sid
   TWILIO_AUTH_TOKEN=your_token
   TWILIO_PHONE_NUMBER=+972501234567
   TWILIO_WHATSAPP_NUMBER=+14155238886
   ```
4. **Restart**: `npm run dev`
5. **שלח הודעה אמיתית!** 🚀

**מדריך מפורט**: `REMINDERS_SETUP.md`

## 📊 מעקב ולוגים

כל הודעה שנשלחת נשמרת ב-`message_logs`:

- מתי נשלחה
- למי (טלפון + שם)
- איזה סוג (SMS/WhatsApp)
- מטרה (תזכורת תור/תשלום/אחר)
- סטטוס (נשלח/נכשל/נמסר/נקרא)
- Provider (twilio/console)
- Message ID מ-Twilio

**שאילתה לדוגמה:**
```sql
SELECT 
  phone_number,
  message_type,
  status,
  sent_at
FROM message_logs
WHERE created_at > now() - interval '7 days'
ORDER BY created_at DESC;
```
